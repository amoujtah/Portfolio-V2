# Moujtahid Adam Portfolio

Cybersecurity portfolio for **Moujtahid Adam**, built with React, Vite, Tailwind CSS, Framer Motion, and Three.js.

## Features

- Interactive 3D cyber card on the home page
- Smooth section navigation
- Projects section with GitHub links and custom covers
- Certificates section with image and PDF previews
- 3D-style tech stack
- Contact cards for LinkedIn, GitHub, CV, and email

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- Supabase integration for optional dynamic project and certificate data

## Getting Started

```bash
git clone https://github.com/amoujtah/moujtahid-adam-portfolio.git
cd moujtahid-adam-portfolio
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```
