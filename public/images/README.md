Images used by the portfolio live here.

Projects:
- Put project screenshots in `public/images/projects/`.
- Use these filenames so the current project cards update automatically:
  - `to-do-list.svg`
  - `portail-absence.svg`
  - `e-commerce.svg`
  - `restaumanager.svg`
  - `cybsera-dashboard.svg`
  - `management-event.svg`

Certificates:
- Put certificate images in `public/images/certificates/`.
- For local certificate cards, add entries in `src/components/ProjectSection.jsx` inside `userCertificates`.
- Each entry can use an image path like `/images/certificates/my-certificate.png`.

Supported formats: `.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`.
